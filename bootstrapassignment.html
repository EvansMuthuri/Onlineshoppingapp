PROFOUNDGRID

A responsive grid system for fixed and fluid layouts. Built in SCSS,
it give you flexibility and fill control.

Responsive

Make grid full of fixed and add permedia layouts as needed. change the columns and gutter widths and remove and add columns as needed.

Semantic

The grid keeps your markup and works without any "grid_x" classes. Unless you want to use them,in which case they are include

Precise

Profound grid usesnegative margins to calculatecolumns. Unlikewith other grid systems,fluid layouts look the same in every browser
